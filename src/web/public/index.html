Frequently asked questions.

Everything you might be wondering, answered.

Is Sentinel free?

Yes — completely free. There's no paywall on any feature. If it's ever hosted at scale, running costs may apply to the host, but the software itself is free.

How do I add Sentinel to my server?

Open the dashboard and use the invite link, or ask the bot owner for the invite. You'll need the Manage Server permission to configure it.

Do I need to log in to use the bot?

No — the bot works in Discord without any login. Logging in to the dashboard just lets you configure everything from a browser instead of slash commands.

What data does Sentinel store?

Only what's needed for features — server settings, economy balances, XP, and moderation logs. We don't read your everyday messages or sell data. See the Privacy Policy.

Why sign in with Google AND Discord?

Google (or email) is an optional first verification step. Discord is what actually connects your account to your servers — it's required to manage them.

Can I self-host it?

No — Sentinel is not available for self-hosting. It runs as a single managed instance so everyone gets the same up-to-date, secure version. Just invite the hosted bot and configure it from the dashboard.

Is my bot token safe?

Your tokens live only in your .env file and are never exposed to the dashboard or browser. Keep that file private and never share it.
