# Custom Emojis 😀

Drop your own **`.png`** files in this folder — they'll automatically show up as
"presets" you can add to any server via:

- the **web dashboard** → Servers tab → *Custom Emojis*
- the **`/emoji preset`** command in Discord

### Tips for good emojis
- **128×128 px**, transparent background (`.png`)
- Keep it **under 256 KB** (Discord's limit)
- Name the file what you want the preset to be called, e.g. `pepe_hype.png` → preset `pepe_hype`
- Use only letters, numbers, and underscores in the filename

The starter set here was generated by `node tools/gen-emojis.js` — re-run that any
time to regenerate them.
